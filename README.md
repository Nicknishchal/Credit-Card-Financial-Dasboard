# Credit Card Financial Analysis Dashboard (Power BI)

## Overview
This project uses Power BI to analyze credit card financial data and generate actionable insights through interactive dashboards. It consists of two dashboards that provide a detailed view of customer behavior and credit card performance.

The objective is to enable data-driven decision-making by visualizing key financial metrics, spending patterns, and customer trends.

---

## Project Components

### 1. Customer Dashboard
This dashboard focuses on customer-related insights and behavioral analysis.

#### Key Insights
- Customer demographics such as age, gender, and income  
- Spending patterns across different segments  
- Customer behavior trends  

---

### 2. Credit Card Dashboard
This dashboard focuses on credit card usage and financial performance.

#### Key Insights
- Credit card usage trends  
- Transaction frequency and amounts  
- Credit limits and utilization  
- Payment behavior and outstanding balances  

---

## Features

- Interactive dashboards for dynamic data exploration  
- Multi-dimensional filtering using slicers  
- Visualization of key financial and behavioral metrics  
- Clear representation of trends and patterns  

---

## Dataset

The dataset includes both transactional and demographic data with the following attributes:

- Customer ID, Age, Gender, Income  
- Transaction amounts and frequencies  
- Credit card types and limits  
- Payment history and balances  

---

## Tech Stack

- Power BI (Dashboard creation and visualization)  
- Power Query (Data cleaning and transformation)  
- CSV / Excel (Data source)
